# Bankly V2

Bankly V2 est une mini-application bancaire interne permettant aux agents de gérer les clients, leurs comptes bancaires et leurs transactions.  
Le projet utilise PHP procédural et une base de données SQL (MySQL/PostgreSQL).

---

## Fonctionnalités

### Authentification
- Login sécurisé pour accéder à l'application
- Logout et gestion de session

### Gestion des clients
- Ajouter, modifier, supprimer et lister les clients
- Informations : nom, email, CIN

### Gestion des comptes bancaires
- Créer, modifier, supprimer et consulter les comptes
- Chaque compte est associé à un client

### Gestion des transactions
- Effectuer des dépôts et retraits
- Historique des transactions par compte

### Dashboard
- Vue d’ensemble des statistiques : nombre de clients, comptes et transactions du jour

### Bonus possibles
- Rôles utilisateurs (admin/agent)
- Recherche, filtres et pagination
- Export PDF/CSV
- Log des connexions
